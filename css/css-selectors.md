# CSS Selectors

## Pseudo-Classes

    selector:pseudo-class {
        property:value;
    }

#### DOM

* `:first-child`
* `:first-of-type`
* `nth-child(n)`

#### Interaction

* `:focus`
* `:hover`

### Anchor Pseudo-Classes

Can set color, background-color, etc.

    /* unvisited link */
    a:link
    
    /* visited link */
    a:visited
    
    /* mouse over link */
    a:hover
    
    /* selected link */
    a:active

If specified in the CSS definition these must be orderd like this!
